# ABC-English-School Webサイト

## 概要
これは、個人のWeb制作ポートフォリオとして作成したレスポンシブ対応のWebサイトです。
友人の実家が英会話教室を運営しており、Webサイトを作りたいという依頼があったと仮定して作成しました。
HTML、CSS、JavaScript、jQuery、を使用してシンプルで美しいデザインを実現しています。
WordPressで作成しています。
